# 🔐 Real-Time Public WiFi Security Analyzer

A cybersecurity tool that analyzes nearby WiFi networks and detects potential threats such as insecure connections, weak encryption, and fake access points (Evil Twin attacks).

---

## 🚀 Features

- 📡 Scan nearby WiFi networks using Python (pywifi)
- 🔐 Detect encryption type (Open, WPA/WPA2)
- ⚠️ Identify insecure and suspicious networks
- 🧠 Evil Twin (fake hotspot) detection
- 📊 Risk score calculation (Safe / Moderate / High Risk)
- 🌐 Real-time web dashboard using Flask
- 🎨 Modern UI with cards, graphs, and color indicators

---

## 🛠 Tech Stack

- Python
- Flask
- HTML, CSS
- pywifi

---
